# Campus-Level Food Ordering & Queue Management System

## Overview
A campus-wide online food ordering system designed to reduce canteen crowding by enabling
pre-ordering, digital payments, time-slot based pickups, QR verification, and real-time order tracking.

## Tech Stack
- Frontend: React.js, Tailwind CSS
- Backend: Spring Boot, Spring Security, JWT
- Database: PostgreSQL, Redis
- Payments: Razorpay (Sandbox)
- Realtime: WebSockets
- Deployment: Docker, AWS, Vercel

## Key Features
- Multi-canteen support
- Time-slot based ordering
- Digital payments
- QR-based food pickup
- Real-time order status
- Admin & staff dashboards
